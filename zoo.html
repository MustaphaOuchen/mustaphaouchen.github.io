ZooManagementSoftware

The idea of this project was to create a software in which the feeding times and food of animals in a zoo can be managed. Moreover, a visitor mode is also implemented so that visitors may check when and where an animal will be fed. A zookeeper and admin mode are also implemented. The zookeeper can check when to feed which animals and when the next feeding must be done. The admin can apply CRUD operations to animals and users (zookeepers and admins) and define relations to animals. It is basically a simple Java application which is connected to a Database via JDBC.
