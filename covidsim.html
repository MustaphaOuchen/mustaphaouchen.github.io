COVID-19 Simulation Project

The idea of this project was to create simulate a virus in an intuitive playful way. This project has been realized with C# and Unity.

With this program one can build his own world and start the spreading of a virus.

One can enable a line graph and/or a barchart to monitor the spreading of the virus in runtime. Values, like how many persons are infected in a day and the r-value, can also be saved in a csv file for further analyses.

One can place three kind of objects in the world and adjust different parameters for each object.

For each object one can monitor the amount of recovered and uninfected persons (green), infected (orange) and infectious (red) people.

Different global simulation parameters can be set. The default values are taken from the Robert Koch-Institut.

Simulation logic

									
The infection event
● Comparison of each person p in the place (venue) with a person i
and each person has a risk of infection r
● p is arbitrary and not infected
● i is infected and is not in hospitalisation
There is a "Carefulnessfactor" c and an "InfectionRiskFactor" r.
"InfectionRiskFactor" v of the point of encounter
c is defined as the average of the "carefulness" of i and p
c and v are mapped to a value between ½ and 2

→ Then the probability of infection is
p(p becomes infected) = r * v * c * g * m
→ g scales the infection event
→ With m ("maskFactor"), the wearing of a mask is taken into account.

Simulation sequence

  • Iterate over all the meeting points and simulate a possible meeting and infection event
  • Add 10 minutes to the simulation date (1.1.2020)
  • Iterate over all living persons of all households
  • Update infection phases and health status of the person under consideration
  • Skip deceased person
  • Take the person to the hospital if necessary
  • Quarantine the person if necessary
  • Take the person home if symptoms are present
  • Try to assign an activity to a person
  • Take person home if they have no activity
  • Test person if activity requires it
