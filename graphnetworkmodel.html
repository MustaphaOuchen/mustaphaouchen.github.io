GraphNetworkModel

CLI-Tool which outputs the number of nodes, number of edges, vertex names, and edge names. The connectivity, diameter, betweenness centrality measure of the graph and the shortest path between a pair or all nodes can be calculated with this program. This program takes a .graphml file as input and can output a custom .graphml file with the calculated information.

Usage

									
Commands:
●	java -jar JavaProgram.jar
	o	Executes the JAR file, an error message will appear.
●	java -jar JavaProgram.jar PathOfInputGraphmlFile  -s  
	o	Executes the JAR file and calculates the shortest path between two vertices. 
●	java -jar JavaProgram.jar PathOfInputGraphmmlFile
	o	Executes the JAR file and calculates:
		▪	Number of nodes
		▪	Number of edges
		▪	Diameter
		▪	Connectivity of the network graph
		▪	NameId attribute for each vertex (the vertex name)
		▪	Edge attributes for each edge (target, id and weight)
●	java -jar JavaProgram.jar PathOfInputGraphmlFile -b 
	o	Betweenness calculation of the program argument and the NameId of the vertex have to be passed. The selected vertex serves as parameter for the switch.
●	java -jar JavaProgram.jar PathOfInputGraphmlFile -a outputfile.graphml
	o	Calculates all properties and saves them in a separate file with the datatype grapml.
